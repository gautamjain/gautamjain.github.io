Android Studio Keyboard Shortcuts

Auto complete

Ctrl + Space Code completion
Ctrl + Shift + Space Smart code completion (by expected type)
Ctrl + Shift + Enter Complete statment (syntax)

Template code

Ctrl + O Template override methods
Ctrl + I Template interface methods

Navigate

Ctrl + N Navigate to class
Ctrl + Shift + N Navigate to file
Ctrl + B Jump to declaration
Alt + Up / Down Jump to previous/next method

Comment

Ctrl + / Toggle line comment
Ctrl + Shift + / Toggle block comment

Select

Ctrl + W Select successively increasing code blocks
Ctrl + Shift + W Decrease current selection to previous state

Search

Shift, Shift Search everywhere
Ctrl + F Find
Ctrl + R Replace
F3 Find next
Shift + F3 Find previous
Ctrl + F6 Refactor

Other

Ctrl + Q Documentation
Alt + Enter Show quick fix
Ctrl + Alt + L Reformat code
Ctrl + Shift + A Lookup IDE commands
